**Injections(单射):**

A->B is an injection if no two inputs have the same output.

|A|<=|B|

**Surjections(满射)**

A->B is a surjection if every output is possible.

|A|>=|B|

**Bijections(双射)**

A->B is a bijection if it is surjection and injection.

|A|=|B|
