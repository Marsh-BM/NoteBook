一.static关键字的核心用法

  用static声明的变量为静态成员变量，也称为类变量。有以下特点

  1.从属于类，只有一份，在类被载入时被显式初始化。

  2.一般用“**类名.类属性**/方法”来调用。

  3.在static方法中不可直接访问非static的成员。

二.静态初始化块

  1.构造方法用于对象的初始化

  2.静态初始化块，用于类的初始化操作

  3.可以初始化static的属性

  4.在静态初始化块中不能直接访问非static的成员。
